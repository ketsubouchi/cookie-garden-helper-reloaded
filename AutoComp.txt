全自動種子コンプ
auto-harvest, auto-plant, and  auto plotを全てONにするだけで
Plotと各条件を自動設定し、種子をコンプリートします

注意1：種子の総コストは数百万CPSになります
注意2：JQBとDuketatorはまとめてリセマラで解禁することをお勧めします

Turn ON auto-harvest, auto-plant, and  auto plot.
Plot and setting will be set automatically to complete all seeds.
Caution 1: Total seed cost is several million CPS.
Caution 2: You should unlock JQB and Duketator by manual savescumming.
Because yhese steps take very long time.

雑草とカビ2種を取得するまでの設定
First stage (weed, brown mold, cromsphore) setting
 Avoid immortal = OFF
 New Seeds = ON
 Mature Seeds = ON
 Check CPS mature = OFF

 Remove weeds = OFF
 Clean Garden = OFF
 Dying plants = ON
 Check CPS dying = OFF

 Avoid buffs = ON
 Rotate soil = OFF (fixed to fertilizer)
 Check CPS plant = OFF
 auto plot = ON

雑草とカビ2種を取得した後の設定
Second stage (After unlock cromsphore) setting
 Avoid immortal = OFF
 New Seeds = ON
 Mature Seeds = ON -> OFF
 Check CPS mature = OFF

 Remove weeds = OFF -> ON
 Clean Garden = OFF -> ON
 Dying plants = ON
 Check CPS dying = OFF

 Avoid buffs = ON
 Rotate soil = OFF (fixed to fertilizer) -> ON (fert/wood)
 Check CPS plant = OFF
 auto plot = ON

表示には反映されませんが、内部はこの設定で動いています
Internally, these setting are used. Not displayed setting.
